# MindPureWellness

MindPureWellness is a professional mental health and wellness website built with **Next.js (Pages Router)**, **Bootstrap 5**, and custom CSS. It features a clean, calming UI designed to provide a supportive experience for individuals seeking mental health services.

## Features
- **Modern & Clean UI**: Calming color palette (light blue, soft white).
- **Fully Responsive**: Built with Bootstrap 5 grid system.
- **Service Catalog**: 9 specialized mental health services with detailed cards.
- **Reusable Components**: Modular Navbar, Footer, and ServiceCard components.
- **Sticky Navigation**: Easy access to sections.
- **Pricing Plans**: Transparent tiered pricing for mental health support.

## Tech Stack
- **Framework**: Next.js (Pages Router)
- **Styling**: Bootstrap 5 + Custom CSS
- **Icons**: Bootstrap Icons
- **Images**: High-quality Unsplash royalty-free images

## Getting Started

### Prerequisites
- Node.js (v14 or later)
- npm (v6 or later)

### Installation
1. Install the dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser to view the website.

## Folder Structure
- `/pages`: Main website pages and app configuration.
- `/components`: Reusable UI components.
- `/styles`: Global and custom CSS styling.
- `/public/images`: Image assets for the project.
